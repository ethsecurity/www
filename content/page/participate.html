--- title: "Participate" ---

ESC invites the public to participate in ESC via IRC, blogs, translations, and other means described below. ESC work with the public through specification reviews as well as contribution of use cases, tests and implementation feedback.

Weekly Gitter meetings

ESC is meeting every Thursday at 17:00 UTC discuss various topics: https://gitter.im/ethsecurity/general.

Code

All source code produced by ESC is published on our Github organization.

Memberships

Organizations join ESC to drive the direction of operational security standards and practices for the Ethereum ecosystem.

ESC is currently working on defining and setting up the membership program.

Questions?

Read our FAQ or contact us for more information about participation and memberships.
