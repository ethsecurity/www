General inquiry

Report a vulnerability

If you have found a vulnerability in an Ethereum project you should first try to contact the maintainer of that project. If the maintainer, however, does not provide any means for secure and responsible disclosure, then you can contact ESC and we will help you with due diligence.

To responsibly disclose a vulnerability: report@ethsecurity.org
